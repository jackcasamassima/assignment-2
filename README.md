DATA 4000: Conditional Logic for Business Decisions
This repository contains a collection of Python programs designed to solve common business decision-making scenarios using conditional logic (if, elif, and else).

Project Structure
The repository includes three distinct Python scripts:

customer_discount.py: Calculates retail discounts based on membership status and purchase volume.

employee_bonus.py: Determines annual bonus amounts based on performance scores and salary brackets.

loan_classifier.py: Classifies loan applicants into risk categories using credit scores and income levels.
